mafw-gst-renderer (0.1.2009.15-1) unstable; urgency=low
  * MAFW gst renderer, pre-release 0.2009.15-1
  * Added transport-actions property. For the moment contains information
    about Seek operation.
  * Unit test disabled by default for system integration purposes.
  * Some tags are detected when Gstreamer is already
    in GST_STATE_PLAYING, so in this case, emit them right away, otherwise

 -- Juha Kellokoski <veitikka6@gmail.com>  Fri, 03 Apr 2009 09:17:14 +0300
mafw-test-gui (0.1.2009.15-1) unstable; urgency=low
  * MAFW, pre-release 0.2009.15-1
  * Disable value-changed handlers for seekbar when
    setting the duration of the current stream, to prevent value-changed
    signals being emitted due to the need to adjust the seekbar value to 
    fit in the new range on EOS, when moving to a new (shorter) stream.
 -- Juha Kellokoski <veitikka6@gmail.com>  Fri, 03 Apr 2009 09:17:14 +0300
mafw-tracker-source (0.1.2009.15-1) unstable; urgency=low
  * MAFW tracker source, pre-release 0.2009.15-1
  * Resolve artist metadata for album categories using Tracker's 
    concat API.
  * Now object identifiers for albums category
